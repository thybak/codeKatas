# CodeKatas
Proyecto que contiene el resultado en código de diversos CodeKatas

## Kata1
Contiene la implementación de 5 soluciones al problema de la búsqueda binaria de un entero en una lista de enteros preordenada:

  * Imperativa
  * Recursiva
  * Troceado imperativo
  * Troceado recursivo
  * Funcional
